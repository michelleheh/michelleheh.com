Web App &
Chrome Extension

dotbind.io

Software Engineer, Product Owner

dotbind.io is a Note taking web app and a chrome extension for programmers. The chrome extention allows users to save code snippets on the fly while browing. The web interface allow users to quick recall prevous capture, with tags simply using the search bar. The web version is enabled with a code editor that provides syntax highlighting. Users can also easily share snippets with any other users in our database.

My contribution to the application

  • Utilized React / Redux to streamline state management resulting in clean and modularized front-end code
  • Implemented a relational and an indexing ElasticSearch database for specialized server responses
  • Incorporated testing and continuous integration to automate build and ensure code maintainability
  • Built a chrome extension to capture code snippets and save to web app for authenticated users
  • Deployed the web application to Amazon Web Service and the Chrome extension to Chrome Web Store

iPhone App - waddle

Grab a lunch buddy!

Software Engineer, Scrum Master

Waddle is an iPhone app that helps you meet new people. Grab lunch someone new – who knows, it may just be the start of a beautiful friendship. The app matches people by location, and finds a restaurant nearby within walking distance. The frond-end is built with React Native, and the back-end is implemented with Node.js and MangoDB.

My contribution to the application

  • Developed front-end code with React-native and flux box, resulting in an intuitive flow on iOS devices
  • Setup server with Node/Express and integrated front end and back end code
  • Incorporated testing and continuous integration to automate build and ensure code maintainability
  • Facilitated team meetings and managed tasks through waffle.io to ensure effective git workflow
